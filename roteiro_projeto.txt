>>> Resumo do Mini Projeto <<< --------------------------------------------
#Objetivo: Desenvolver um relatório em PowerBI com Insights sobre Feedback de Usuários, utilizando PostgreSQL, PowerBI, e Modelagem de Dados.

#Sequência
1º Criar um servidor de banco dados PostgreSQL
2º Criar as tabelas que receberão os dados
3º Transferir os dados da planilha para o banco de dados
4º Conectar o Power BI ao banco de dados
5º Modelar os dados
6º Planejar o Relatório
7º Criar o relatório
8º Conectar o PowerBI ao Serviço WEB
9º Publicar o Relatório
